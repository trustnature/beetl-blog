# beetl-blog

#### 项目介绍
beetl实现的一个  小 blog，有配套的视频，用于新手一起学习 beetl、beetlSQL

#### 软件架构
Beetl + BeetlSql +SpringBoot2 等
数据库：mysql5.7
数据库脚本在根目录下的 数据库脚本.sql中
